Pseudoelemento :empty

También existe el Pseudoelemento empty, el cual hace referencia a los elementos qu están vacios.

Se puede combinar con otros, como por ejemplo:

div:not(:empty){
    ...
}

El cual afecta a todos los divs que no estén vacíos.
